About StockFlow

Project Overview

StockFlow is a stock portfolio management and tracking system developed for CS3620 Database Systems course.

This is Checkpoint 1 - a minimal working version demonstrating database design and basic functionality.

Features

  • Track 500+ stocks from the S&P 500 index
  • View historical stock price data
  • Create and manage investment portfolios
  • Maintain a watchlist of interesting stocks
  • Analyze market trends with interactive analytics

Database Schema (10 Tables)

  1. users - User accounts and authentication
  2. portfolios - Investment portfolios
  3. sectors - Stock industry sectors
  4. stocks - Stock master data
  5. stock_prices - Historical daily prices
  6. transactions - Buy/sell records
  7. holdings - Current portfolio holdings
  8. watchlist - User watchlists
  9. alerts - Price alerts
  10. audit_log - Activity tracking

Data Sources (3 Public Datasets)

  1. S&P 500 Companies - Company information and sectors (Kaggle)
  2. Stock Prices - Historical daily prices (Yahoo Finance via yfinance)
  3. S&P 500 List - Current constituent companies (Wikipedia)

Technology Stack

  • Backend: Python, Flask
  • Database: MySQL
  • Frontend: HTML, CSS
  • Data: pandas, yfinance, kagglehub

Interactive Features

  • Write Operations: Add/remove stocks from watchlist
  • Read Operations: View stocks, portfolios, prices, analytics
  • Analytics: Sector distribution, top volume stocks, volatility analysis
